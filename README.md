# 블로그팀 — 여성골격 부서 패키지

## 파일 구성
- `IamI_카드뉴스_템플릿.pptx` : 세로형(4:5) 9장 카드뉴스 템플릿 (전 체형 공용)
- `cardnews.py` : 카드뉴스 자동 생성 스크립트 (spec.json + 사진 8장 → 1080x1350 JPG 9장)
- `guidelines/` : 체형별 블로그 글쓰기 지침 3종 (스트레이트/웨이브/내추럴 — 보이스·글 구조·지식 베이스)

## 워크플로우
1. "체형/주제 글작성" → 지침 기반 블로그 글 + SEO 제목 후보
2. 발행 글 복붙 + 사진 8장(01=커버 클린 원본, 02~08=본문) → 카드뉴스 9장 + 인스타 캡션 + 네이버 클립 글

## cardnews.py 기술 사양
- 밑줄 바: 2-패스 픽셀 실측(1차 렌더 → 핑크 EC5FB8 실측 → 괄호 아래 갭 7px 재배치 → 2차 렌더)
- 해시태그 필: 가변 폭(텍스트폭+0.378in) 2열 자동 배치, 최대 10개
- 커버: 은은한 다크 그라데이션(최대 45%) 자동 — 텍스트 박힌 이미지 사용 금지
- 폰트: 페이퍼로지 9종 (iami-cn-k7x92m 리포에서 /root/.fonts/ 설치)
